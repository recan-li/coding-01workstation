# 概述

本目录主要是DNS解析的简要测试。

# 文件说明

```shell
└── dns
    ├── build.sh 			编译脚本
    ├── dns.c 				域名解析的组包/解析源码
    ├── dns.h 				导出接口的头文件
    ├── main.c 				测试源代码的main.c
    └── README.md 			本说明文档
```

# 执行编译

在dns目录下执行：

```c
./build.sh
```

# 编译清除

在dns目录下执行：

```c
./build.sh clean
```
